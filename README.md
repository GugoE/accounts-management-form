# Vue Accounts Management

Тестовое задание для Vue Frontend Developer.  
Реализована форма управления учетными записями с поддержкой тегов, валидацией и сохранением в Pinia + localStorage.

## Стек

- Vue 3 + Composition API
- TypeScript
- Pinia (стейт менеджер)
- Element Plus (UI-фреймворк)

## Функционал

- Добавление/удаление учетных записей
- Поля учетной записи:
    - **Метки** — необязательное поле, ввод через `;`, отображается как теги, максимум 50 символов
    - **Тип записи** — LDAP / Локальная, скрывает/показывает поле Пароль
    - **Логин** — обязательное поле, максимум 100 символов
    - **Пароль** — обязательное поле для локальной учетной записи, максимум 100 символов
- Валидация обязательных полей при потере фокуса или изменении селекта
- Сохранение всех данных в Pinia + localStorage, чтобы данные сохранялись при обновлении страницы
